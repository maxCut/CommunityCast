This is the sender application

Instructions for starting the server

1 Navigate to the CommunityCastReceiver folder on your computer

2 Run the command $ node index.js

3 The server runs index.html on port 3000

Instructions for sending messages

1 Connect to the wifi name "builds-sec", the password is written on the whiteboard

2 Click on the three dots in the upper right corner of the screen

3 Click on "Cast..."

4 Select the chromecast name "Hank the Tank" and wait until it is successfully connected

5 Once you are connected, right click and select "Inspect" in the pop-up window

6 Navigate to the "Console" tab

6 Send messages to the display by using the function sendMessage("some string")
